project is resizing photos

for main visit:
http://localhost:5000/

for starting a resize:
 1- http://localhost:5000/resizeimg?name=ddd&height=200&width=300
 2- http://localhost:5000/resizeimg?name=ddd&height=200&width=a


for build:
npm run build

for start server:
npm run start

for checking test:
npm run test
